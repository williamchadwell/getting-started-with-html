What makes the Web Work?

The web is a bunch of computers that communicate with one other. When a person goes to a web page like www.yahoo.com, their computer sends a HTTP Request to a server. The server finds the appropriate HTML document and sends it back to the user's computer where a web browser interprets the page and displays it on the user's screen.

What is HTML?

HTML (HyperText Markup Language) - HTML documents form the majority of the content on the web. HTML is made up of text content which is "what you see" as well as markup which describes "what it actually looks like".

Elements and Tags-

HTML documents are made up of HTML elements ( contents ) This whole thing is called an element. When writing HTML, we tell browsers the type of each element by using HTML tags.

Do some people consider Computers are Stupid? YES

Computers are stupid because they interpret instructions literally. This makes them very unforgiving since a small mistake by a programmer will cause huge problem within a program.

Differences between block and inline elements-

HTML elements are fall under two categories: inline or block. Block elements form whats called an "invisible box" around the content inside of them.
